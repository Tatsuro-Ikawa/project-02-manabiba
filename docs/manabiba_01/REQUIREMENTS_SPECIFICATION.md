## 1. システム概要

### 1.1 システム名
**人生学び場 こころ道場**

### 1.2 システムの目的
AI×セルフマネジメント×伴走コーチングで、「なりたい自分」へ確実に歩むための場を提供する。

### 1.3 システムの概要
ユーザーが自己理解を深め、目標を設定し、PDCAサイクルを通じて継続的に成長をサポートするWebアプリケーション。

### 1.4 主要な価値提案
- **自己理解**: 価値観、強み、願望、課題を明確化
- **目標設定**: SMARTフレームワークに基づいた具体的な目標設定
- **PDCAサイクル**: 計画→実行→確認→改善の継続的なサイクル
- **伴走コーチング**: AIとコーチによる継続的なサポート

### 1.5 対象ユーザー
- **プライマリユーザー**: 自己成長を目指す個人
- **セカンダリユーザー**: コーチ、管理者

## 4. データモデル

### 4.1 主要エンティティ

#### 4.1.1 UserProfile（ユーザープロファイル）
```typescript
{
  uid: string;
  email: string;
  displayName: string;
  photoURL?: string;
  role: 'user' | 'coach' | 'senior_coach' | 'admin';
  subscription: SubscriptionInfo;
  createdAt: Date;
  updatedAt: Date;
  lastLoginAt: Date;
}
```

#### 4.1.2 PDCAData（PDCAデータ）
```typescript
{
  id?: string;
  uid: string;
  date: string;
  plan?: string;
  do?: string;
  check?: string;
  action?: string;
  createdAt?: Timestamp;
  updatedAt?: Timestamp;
}
```

#### 4.1.3 SMARTGoal（SMART目標）
```typescript
{
  id: string;
  uid: string;
  themeId?: string;
  themeTitle?: string;
  userType: 'aspiration' | 'problem';
  specificDescription: string;
  measurementValue: number;
  measurementUnit: string;
  measurementFrequency: string;
  relevanceReason: string;
  relatedValues: string[];
  targetDate: Date;
  milestones: Milestone[];
  status: 'draft' | 'active' | 'completed' | 'paused' | 'cancelled';
  progress: number;
  createdAt: Date;
  updatedAt: Date;
}
```

#### 4.1.4 ThemeSelectionData（テーマ選択データ）
```typescript
{
  id: string;
  userType: 'aspiration' | 'problem';
  entries: ThemeEntry[];
  selectedTheme?: string;
  completed: boolean;
  status: 'draft' | 'published';
  currentStep: ThemeSelectionStep;
  createdAt: Date;
  updatedAt: Date;
}
```

### 4.2 データストレージ

| データ種別 | ストレージ | 説明 |
|-----------|-----------|------|

| PDCAデータ | Firestore: pdca_entries | PDCAエントリ |
| SMART目標 | Firestore: users/{uid}/smart-goals | SMART目標 |

## 7. 制約事項

### 7.1 技術的制約

- **ブラウザ対応**: モダンブラウザ（Chrome, Firefox, Safari, Edge）
- **デバイス**: PC、タブレット、スマートフォン
- **ネットワーク**: インターネット接続必須
- **Firebase制約**: Firestoreのセキュリティルールに準拠

### 7.2 ビジネス制約

- **サブスクリプション**: 3つのプラン（free, standard, premium）
- **機能制限**: プランに応じた機能制限
- **データ保持**: ユーザーが削除するまでデータを保持

### 7.3 法的制約

- **個人情報保護**: 個人情報保護法に準拠
- **データ暗号化**: 機密情報は暗号化して保存
