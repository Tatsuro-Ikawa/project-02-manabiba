## 1. カラーパレット

**参考**: 自己変革7日間プログラムチラシ（PNG画像から抽出）

## 5. アイコン

### 5.1 アイコンライブラリ

#### 5.1.1 実装版の標準：Google Fonts 無料アイコン（Material Symbols）

実装版では **Google Fonts の Material Symbols**（無料）をアイコンとして使用する。

- **提供元**: [Google Fonts – Material Symbols](https://fonts.google.com/icons)（無料・ライセンス問題なし）
- **読み込み例**（HTML の `<head>` または CSS の `@import`）:
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@24,400,0,0" rel="stylesheet" />
  ```
- **使い方**: フォントの ligature でアイコン名を指定する。
  ```html
  <span class="material-symbols-outlined" aria-hidden="true">home</span>
  ```
- **スタイル例**（サイズ・色）:
  ```css
  .material-symbols-outlined {
    font-size: 24px;
    color: var(--color-primary);
  }
  ```
- **アイコン一覧・名前**: 上記 Google Fonts のアイコンページで検索し、表示名（英語）をそのままテキストとして使う（例: `home`, `person`, `menu`, `expand_more`）。
- **プロトタイプとの対応**: プロトタイプでは絵文字（🏠 など）や簡易アイコンを使用しているが、実装時は同じ意味の Material Symbols の名前で置き換える。サイドバー等の対応例は下表のとおり。

| プロトタイプ（例） | 意味 | Material Symbols 名 |
|-------------------|------|---------------------|
| 🏠 | ホーム | `home` |
| 🌱 | スタート | `rocket_launch` または `play_circle` |
| 🎯 | ゴール | `flag` または `track_changes` |
| 📊 | 計画・記録 | `bar_chart` または `assignment` |
| 📝 | 実行・反省 | `edit_note` または `checklist` |
| ✍️ | 反省 | `rate_review` または `edit` |
| 👤 | マイページ | `person` |
| ☰ | メニュー | `menu` |

#### 5.1.2 その他のライブラリ（必要に応じて）

- **Material-UI Icons**（`@mui/icons-material`）: コンポーネントで使いたい場合。Material Design 系で名前は Material Symbols とほぼ一致する。
- **Lucide React**（`lucide-react`）: モダンな見た目のアイコンが必要な場合。標準は 5.1.1 に統一し、必要時のみ併用する。
