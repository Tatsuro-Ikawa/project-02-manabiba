# PDFファイル処理ガイド

## 📋 PDFファイルの認識について

### 現在の制限

**直接的なPDF読み取り**: ❌ 不可能
- PDFファイルを直接開いて内容を読み取ることはできません
- バイナリ形式のため、テキストとして直接読み取れません

### 可能な方法

以下の方法でPDFの内容を扱うことができます：

---

## 1. PDFからテキストへの変換

### 1.1 オンラインツール（推奨）

#### Adobe Acrobat Online
- **URL**: https://www.adobe.com/jp/acrobat/online/pdf-to-txt.html
- **機能**: PDFをテキストファイルに変換
- **特徴**: 無料、高精度

#### SmallPDF
- **URL**: https://smallpdf.com/jp/pdf-to-txt
- **機能**: PDFをテキストに変換
- **特徴**: 無料、簡単

#### iLovePDF
- **URL**: https://www.ilovepdf.com/ja/pdf_to_txt
- **機能**: PDFをテキストに変換
- **特徴**: 無料、多機能

### 1.2 デスクトップアプリケーション

#### Adobe Acrobat
- **機能**: PDFをテキストにエクスポート
- **特徴**: 有料、高精度

#### PDFelement
- **機能**: PDFをテキストに変換
- **特徴**: 有料、多機能

### 1.3 コマンドラインツール

#### pdftotext（Linux/Mac）
```bash
pdftotext input.pdf output.txt
```

#### Python（pdfplumber）
```python
import pdfplumber

with pdfplumber.open("input.pdf") as pdf:
    text = ""
    for page in pdf.pages:
        text += page.extract_text()
    
    with open("output.txt", "w", encoding="utf-8") as f:
        f.write(text)
```

## 2. PDFの内容を共有する方法

### 方法1: テキストファイルに変換して共有

1. PDFをテキストファイル（`.txt`）に変換
2. テキストファイルをプロジェクトに追加
3. 私がテキストファイルを読み取って分析

**手順**:
```bash
# テキストファイルをプロジェクトに追加
# 例: docs/manabiba_01/paper_based_tools/
#   - text_content.txt
#   - worksheet_content.txt
#   - checklist_content.txt
```

### 方法2: 内容をコピー&ペースト

1. PDFを開く
2. 内容をコピー
3. 新しいMarkdownファイルに貼り付け
4. 私がMarkdownファイルを読み取って分析

**手順**:
```markdown
# 紙ベースツールの内容

## テキスト（紙レベル）
[PDFからコピーした内容をここに貼り付け]

## ワークシート
[PDFからコピーした内容をここに貼り付け]

## チェックリスト
[PDFからコピーした内容をここに貼り付け]
```

### 方法3: スクリーンショット + OCR

1. PDFのスクリーンショットを撮る
2. OCRツールでテキスト化
3. テキストファイルとして保存

**OCRツール**:
- **Google Drive**: PDFをアップロードして、Googleドキュメントで開く（自動OCR）
- **Adobe Acrobat**: OCR機能付き
- **オンラインOCR**: https://www.onlineocr.net/

## 5. PDFの種類による対応

### 5.1 テキストベースのPDF

**特徴**: テキストが選択可能
**方法**: 
- 直接コピー&ペースト可能
- オンラインツールで変換可能

### 5.2 画像ベースのPDF（スキャン）

**特徴**: テキストが選択不可
**方法**: 
- OCRツールが必要
- Google DriveのOCR機能を推奨

### 5.3 フォーム付きPDF

**特徴**: 入力欄がある
**方法**: 
- フォームの構造を手動で記録
- スクリーンショット + 説明文

## 8. 注意事項

### 8.1 文字化け

**問題**: PDFからテキストに変換した際、文字化けが発生する場合

**対処法**:
- UTF-8エンコーディングで保存
- 日本語対応のツールを使用
- 必要に応じて手動で修正

### 8.2 レイアウトの保持

**問題**: PDFのレイアウトが失われる場合

**対処法**:
- 重要なレイアウト情報は手動で記録
- スクリーンショットを併用
- Markdownで構造化して記録

### 8.3 画像・図表

**問題**: PDF内の画像や図表が変換されない

**対処法**:
- 画像は別途保存
- 図表は説明文で補足
- 必要に応じて再作成
